# 配置卖家授权

如果您的应用需要访问国际站卖家的业务数据，那么您的应用需要获得卖家的授权，您需要引导他们完成“使用卖家账号登录并授权”的流程。

授权策略
阿里巴巴国际站开放平台采用OAuth 2.0协议进行用户认证，并针对不同的应用分类提供了不同的授权策略，这些授权策略是针对您的应用所属的应用分类预先定义的。您可以在授权管理页面的上方查看到您应用的详细授权策略，包括以下信息：

●授权策略：你的应用程序的总体策略
●访问令牌有效期：您应用程序的访问令牌的有效期
●Refresh Token Duration：刷新令牌的有效期（详细信息请参考OAuth文档）
● Authorized Page：是否显示授权页面（默认显示授权页面）
●授权协议：OAuth2.0授权类型 
●授权用户限制：可以授权该应用程序的卖家的最大数量

以下3种授权策略分别适用于不同的APP类别：
●允许绑定用户授权：关于此授权策略，只有在“授权卖家白名单”字段中配置的卖家才可以授权您的应用。有关详细信息，请参阅以下部分。
●允许登录用户授权：此策略下，任何国际站卖家都可以授权您的应用，所以您无需配置卖家白名单，只需将您的授权URL发送给您的卖家进行授权即可。
●允许订购授权：关于此授权策略，需要在外贸服务市场下单，并通过订单中的授权链接进行授权。

指定卖家白名单
如果您开发的应用是针对特定卖家的，您可以通过在阿里巴巴国际站开放平台上指定卖家白名单来限制授权范围，只有这些卖家才能授权您的应用访问他们在阿里巴巴国际站的业务数据。您的应用部署上线后，您需要将授权URL提供给卖家。卖家首次登录您的应用时，系统会提示他们完成授权流程。
按照以下步骤指定可以授权您的应用程序访问其业务数据的卖家的短代码。
●打开APP控制台，点击应用管理->授权管理。
●在授权信息部分，查看应用程序的授权策略描述。
●在授权卖家白名单部分，在授权字段输入卖家账户，多个卖家账户可以用分号分隔。
●点击提交。
注：如果您是第三方开发者（服务商身份），您的应用开放给国际站卖家，那么在应用上线部署时，您需要将应用发布至外贸服务市场，卖家将通过订购后，即可完成授权流程。
